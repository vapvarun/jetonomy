## Post 2 — Feature Spotlight: Trust Levels (Infographic, 1 image)

### Image Brief

Vertical infographic. Title: "How Trust Levels Work in Jetonomy."

Six rows, one per trust level (0–5). Each row:
- Level number (colored circle: grey for 0, green gradient increasing toward 5)
- Level name (Newcomer, Basic, Regular, Leader, Moderator, Trusted Moderator)
- 1-2 key capabilities or restrictions

Key data points to include:
- Level 0: 3 posts/day max, no links
- Level 1: 10 posts/day, links still restricted
- Level 2: Links allowed
- Level 3: Edit freely, follow topics
- Level 4: Moderate content, move posts (manually assigned)
- Level 5: Trusted mod (manually assigned)

Bottom callout: "Levels 0–3 earned automatically. Levels 4–5 assigned by you."

Clean, readable at mobile size. No more than two fonts.
